# Image-Similarity-Search-Milvus
Image Similarity search build on Milvus 

## 1. Getting Started

Clone the repo:

  ```bash
  git clone https://github.com/bigmb/Image-Similarity-Search-Milvus.git
  ```
  
## 2. Requirements

```
python>=3.6
numpy
pymilvus==2.0.0rc7
tensorflow
pandas
glob
argparse
cv2
pathlib
PIL

```
Install all dependent libraries:
  ```bash
  pip install -r requirements.txt
  ```
